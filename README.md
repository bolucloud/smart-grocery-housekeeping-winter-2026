## Branch & PR Naming Conventions
### Branch Prefixes
1. Feature Branches: These branches are used for developing new features. Use the prefix `feature/{Story Name}`. 
2. Bugfix Branches: These branches are used to fix bugs in the code. Use the prefix `bugfix/{Bug Name}`.
3. Test Branches: These branches are made for any automation test fixures. Use the prefix `test/{Test Name}-`.
